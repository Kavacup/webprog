Why so?

Programmers use "Hello, World!" As the first program in a new language, to make sure that the development environment is correct and the code is compiled without errors.
This is a simple way to check how the syntax of text output on the screen works. It also became a tradition and a ritual in programming, symbolizing the beginning of studying a new language.

The first program "Hello, World!" She appeared in the book “The C Programming Language” (1978), written by Brian Kernigan and Dennis Ritchi, creators of Code’s language. In the book, the code showed how to bring the text on the screen, and they chose the phrase “Hello, World!” Just because it was short and positive.
Since then, this has become a tradition in programming.
